# MoodSpark
ENG:

MoodSpark

Idea:
A small application that helps a person quickly record their mood and receive a small "spark" of support.

How it works:

The user chooses a mood (happy / tired / anxious / inspired, etc.)

The application shows:

a short motivational phrase

a pleasant color or gradient

a mini-idea: “take a break”, “listen to music”, “write one thought”

The entire history is stored as a simple calendar or feed

UKR:

MoodSpark

Ідея:
Невеликий додаток, який допомагає людині швидко зафіксувати настрій і отримати маленьку «іскру» підтримки.

Як це працює:

Користувач обирає настрій (happy / tired / anxious / inspired тощо)

Додаток показує:

коротку мотиваційну фразу

приємний колір або градієнт

міні-ідею: “take a break”, “listen to music”, “write one thought”

Вся історія зберігається у вигляді простого календаря або стрічки
